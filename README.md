# 🌦️ Django Weather App

Bu loyiha Django yordamida qurilgan oddiy, sodda va tezkor ob-havo tekshirish web-ilovasi.
OpenWeatherMap API orqali joriy ob-havo ma'lumotlarini ko‘rsatadi.

---

## 🚀 Funksiyalar

- Shahar nomi bo‘yicha ob-havo qidirish
- Harorat, namlik, shamol tezligi, holat (weather description)
- Ob-havo ikonlari bilan ko‘rsatish
- Xatoliklarni to‘g‘ri qaytarish (404, noto‘g‘ri shahar, API xatolari)
- .env orqali maxfiy konfiguratsiya
